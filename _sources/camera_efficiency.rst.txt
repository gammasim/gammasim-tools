.. _CameraEfficiency:

Camera Efficiency
=================

In this section you find the reference documentation of the Camera Efficiency modules.
The main functionality of this module is to provide an interface to the testeff program
provided by simtel_array. A Telescope Model (see :ref:`telescope_model`) and few more
physical parameters (zenith angle, etc) are required inputs.
