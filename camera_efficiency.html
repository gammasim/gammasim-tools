Camera Efficiency¶

In this section you find the reference documentation of the Camera Efficiency modules. The main functionality of this module is to provide an interface to the testeff program provided by simtel_array. A Telescope Model (see telescope_model) and few more physical parameters (zenith angle, etc) are required inputs.

If True, existing results files will be removed and analysis will be done again.
